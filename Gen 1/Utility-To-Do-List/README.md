# Minimalist To-Do List 📝

A clean, fast, and minimalist to-do list application for managing your daily tasks effectively without unnecessary clutter.

## 🌟 Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Minimalist user interface
- Local state management

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started
Open `index.html` in your web browser to start organizing your tasks!
