# Minimalist Clock 🕒

A beautiful, sleek, and highly functional web-based clock utility. Designed with minimalism in mind, this clock provides a clean interface for checking the time.

## 🌟 Features
- Real-time clock display
- Minimalist and modern UI
- Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started
Simply open the `index.html` file in your preferred web browser to run the application locally. No installation required!
