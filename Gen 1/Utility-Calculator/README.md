# Web Calculator 🧮

A clean, responsive, and fully functional web-based calculator. Designed with a premium aesthetic and accurate calculation logic.

## 🌟 Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Sleek and modern UI design
- Fully responsive layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started
Simply open the `index.html` file in your preferred web browser to run the calculator locally.
